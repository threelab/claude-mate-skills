## 故事继续：餐厅出餐

你的餐厅已经能接单了。但还有一个问题：

**厨师做的菜，顾客不一定满意。**

比如：
- 顾客说"不要辣"，厨师可能还是放了一点辣椒
- 顾客说"少放盐"，厨师可能没有完全理解

所以你需要一个**"出餐检查"**：
- 厨师做好菜 → 端给顾客看 → 顾客说"这样可以"或者"再调整一下"
- 调整完成后，才真正上菜

在我们的口播剪辑工具里，这个"出餐检查"就是**审核页面**。

## 怎么实现审核页面？

审核页面有两个核心部分：

### 第一部分：数据准备

程序需要先处理完视频，生成**审核数据**：

```javascript
// 这是处理完视频后，要给页面看的数据
const reviewData = {
  // 视频信息
  videoFileName: '2026年学习计划.mp4',

  // 每个字的信息（从语音识别结果来）
  words: [
    { text: '嗯',    begin: 0.0, finish: 0.2, selected: true,  type: 'filler' },
    { text: '那',    begin: 0.2, finish: 0.4, selected: true,  type: 'filler' },
    { text: '个',    begin: 0.4, finish: 0.6, selected: true,  type: 'filler' },
    { text: '大',    begin: 0.8, finish: 1.0, selected: false, type: 'normal' },
    { text: '家',    begin: 1.0, finish: 1.2, selected: false, type: 'normal' },
    { text: '好',    begin: 1.2, finish: 1.5, selected: false, type: 'normal' },
    // ... 更多内容
  ],

  // 统计信息
  totalDuration: 330,     // 原始时长（秒）
  totalSelectedDuration: 45,  // 选中要删除的时长
  totalWords: 520,        // 总字数
  selectedWords: 38       // 被标记删除的字数
};
```

### 第二部分：页面展示

用 HTML 把数据变成用户能操作的界面。

**简化的页面代码：**

```html
  </style>
</head>
<body>
  <div class="review-container">
    <!-- 左边：视频播放器 -->
    <div class="video-section">
      <video id="videoPlayer" controls autoplay>
        <source src="video.mp4" type="video/mp4">
      </video>

      <div class="stats">
        <p>原始时长: <span id="totalDuration">0</span> 秒</p>
        <p>预计删除: <span id="selectedDuration">0</span> 秒</p>
      </div>

      <div class="actions">
        <button class="btn btn-secondary" id="selectFillers">全选语气词</button>
        <button class="btn btn-secondary" id="clearAll">取消全选</button>
        <button class="btn btn-primary" id="generateClip">生成剪辑</button>
      </div>
    </div>

    <!-- 右边：文本内容 -->
    <div class="text-section">
      <h3>点击字词切换是否删除</h3>
      <div id="wordsContainer"></div>
    </div>
  </div>

  <script>
    // 1. 从后端拿数据（这里简化成直接写在代码里）
    const reviewData = { ... }; // 上面定义的数据

    // 2. 把文字显示到页面上
    function renderWords() {
      const container = document.getElementById('wordsContainer');
      container.innerHTML = '';

      reviewData.words.forEach((word, index) => {
        const item = document.createElement('span');
        item.className = 'word-item';

        if (word.type === 'filler') item.classList.add('filler');
        if (word.selected) item.classList.add('selected');

        item.textContent = word.text;

        // 点击切换选中状态
        item.addEventListener('click', () => {
          word.selected = !word.selected;
          item.classList.toggle('selected');
          updateStats();
        });

        container.appendChild(item);
      });

      updateStats();
    }

    // 3. 更新统计信息
    function updateStats() {
      const selectedDuration = reviewData.words
        .filter(w => w.selected)
        .reduce((sum, w) => sum + (w.finish - w.begin), 0);

      document.getElementById('totalDuration').textContent =
        reviewData.totalDuration.toFixed(1);
      document.getElementById('selectedDuration').textContent =
        selectedDuration.toFixed(1);
    }

    // 4. 按钮功能
    document.getElementById('selectFillers').addEventListener('click', () => {
      reviewData.words.forEach(w => {
        if (w.type === 'filler') w.selected = true;
      });
      renderWords();
    });

    document.getElementById('clearAll').addEventListener('click', () => {
      reviewData.words.forEach(w => w.selected = false);
      renderWords();
    });

    document.getElementById('generateClip').addEventListener('click', async () => {
      // 告诉后端：按当前选择来生成剪辑
      const response = await fetch('/api/generate-clip', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          selectedWords: reviewData.words.filter(w => w.selected)
        })
      });

      const result = await response.json();
      if (result.success) {
        alert('剪辑完成！文件位置：' + result.outputFile);
      }
    });

    // 启动时渲染
    renderWords();
  </script>
</body>
</html>
```

## 真正生成剪辑视频

审核确认完，就要真正生成剪辑后的视频了。

**逻辑：**

1. 从用户的选择里，找出**哪些时间段要保留**

**实际上，剪辑视频是个复杂的过程，需要处理：**

| 问题 | 处理方式 |
|------|---------|
| 时间段的边界 | 让保留段 and 删除段平滑过渡，避免"跳"的感觉 |
| 多个删除段 | 把相邻的删除段合并，避免切成太多小片段 |
| 音视频同步 | 确保音频和视频对齐，不要"嘴型对不上声音" |
| 文件格式 | 保持和原始视频相同的编码、分辨率 |

## 完整的流程：用户用一遍看看

**步骤 1：选择视频**
1. 双击启动脚本
2. 浏览器打开界面
3. 点击"浏览"，选择视频文件

**步骤 2：处理视频**
1. 点击"开始处理"
2. 程序提取音频 → 语音识别 → 分析内容
3. 完成后显示"打开审核页面"按钮

**步骤 3：审核确认**
1. 打开审核页面

**步骤 4：生成剪辑**
1. 确认无误后，点击"生成剪辑"
2. 程序调用 FFmpeg，真正剪辑视频

1. 用户打开输出文件夹，看到剪辑后的视频

## 让工具更好用：细节优化

### 优化 1：快速跳转

用户点某个词，视频自动跳到对应的位置播放：

```javascript
// 在审核页面里
item.addEventListener('click', () => {
  // 切换选中状态（之前的逻辑）
  word.selected = !word.selected;
  item.classList.toggle('selected');
  updateStats();

  // 新增：视频跳转到对应的位置
  const video = document.getElementById('videoPlayer');
  video.currentTime = word.begin;  // 跳到这个字开始的位置
  video.play();                    // 开始播放
});
```

### 优化 2：批量操作

用户不想一个个点，提供快捷键：

```javascript

    // 自动跳到下一个词
  }

  }

```

### 优化 3：智能分组

程序自动把**相邻的语气词合并成一段**，用户只要点一次就能选/不选一组：

```javascript
// 分析词序列，把相邻的、类型相同的词分成一组
// [嗯, 那个, 大家, 好, 啊, 今天]
//   ↓
// 组1：[嗯, 那个]（语气词，可删除）
// 组2：[大家, 好]（正常内容）
// 组3：[啊]（语气词，可删除）
// 组4：[今天, ...]（正常内容）

function groupWords(words) {
  const groups = [];
  let currentGroup = [];
  let currentType = null;

  for (const word of words) {
    if (word.type !== currentType) {
      // 类型变了，结束上一组，开始新组
      if (currentGroup.length > 0) {
        groups.push({ words: currentGroup, type: currentType });
      }
      currentGroup = [word];
      currentType = word.type;
    } else {
      // 类型相同，加入当前组
      currentGroup.push(word);
    }
  }

  // 不要忘了最后一组
  if (currentGroup.length > 0) {
    groups.push({ words: currentGroup, type: currentType });
  }

  return groups;
}
```

### 优化 4：自动保存

用户选了一半，刷新页面，选择丢失了？不行，要保存：

```javascript
// 每次修改选择，保存到浏览器的 localStorage
function saveSelection() {
  localStorage.setItem(
    'review_selection_' + videoFileName,
    JSON.stringify(reviewData.words.map(w => w.selected))
  );
}

// 页面加载时，读取之前的选择
function loadSelection() {
  const saved = localStorage.getItem('review_selection_' + videoFileName);
  if (saved) {
    const selections = JSON.parse(saved);
    reviewData.words.forEach((w, i) => {
      w.selected = selections[i] || false;
    });
  }
}
```

## 你可能会问的问题

**Q: 生成剪辑视频要花很长时间吗？**

A: 取决于视频长度和电脑性能。一般来说，剪辑一个5分钟的视频，FFmpeg 处理几十秒到几分钟。

**Q: 剪辑后的视频画质会变差吗？**

A: 如果用"直接复制"（copy codec）而不是重新编码，画质不会变差，速度也快。但删除操作可能需要重新编码。

**Q: 审核页面的数据存在哪里？**

A: 处理完视频后，程序会把数据保存成 JSON 文件，放在 `output/` 目录下。审核页面读取这个文件。

**Q: 我能把这个工具做成网页版，让别人在线用吗？**

A: 可以，但要注意：需要服务器、带宽、用户上传视频的存储、隐私问题。这比本地运行要复杂很多。
