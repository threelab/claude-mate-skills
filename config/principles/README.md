# 用户习惯

用户个人偏好，AI 审核时参考。

## 文件清单

| 文件 | 类型 | 内容 |
|------|------|------|
| 1-core.md | 原则 | 删前保后（最高优先级） |
| 2-语气词检测.md | 偏好 | 嗯啊呃 + 删除边界 |
| 3-静音段处理.md | 阈值 | ≤0.2s忽略, 0.2-1s预选删, >1s建议删 |
| 4-重复检测.md | 偏好 | 句间重复 + 句内重复 + 重说纠正（合并） |
| 5-卡顿词.md | 偏好 | 那个那个、就是就是 |
| 7-连续语气词.md | 偏好 | 嗯啊、啊呃 |
| 9-残句检测.md | 偏好 | 话说一半被打断 |

## AI 审核顺序（按优先级）

1. **静音 >1s** → 建议删
2. **残句** → 删整句（话说一半 + 静音）
3. **重复** → 删前面，保后面（句间/句内/纠正，见4-重复检测.md）
4. **卡顿词** → 删前面（那个那个、就是就是）
5. **语气词** → 标记待人工确认（嗯啊呃）

## 核心原则

**删前保后**：口播过程中，后说的通常更完整、更准确，删除前面（口误/残句/重复的第一次），保留后面（完整句/纠正后的/重复的最后一次）。
