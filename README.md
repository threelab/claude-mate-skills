# 口播视频智能剪辑 (Clip Announcement)

这是一个基于火山引擎 ASR 接口的智能视频剪辑工具，旨在帮助用户快速识别并定位口播视频中的语气词、错误信息及无效内容。

## 🚀 项目结构

项目采用分层架构设计，逻辑清晰，易于扩展：

- **`core/`**: 核心逻辑层，包含任务规划器 (`planner.js`) 和转录文本分析器 (`analyst.js`)。
- **`tools/`**: 工具箱，包含音频提取脚本 (`ffmpeg_wrapper.sh`) 和 ASR 接口封装 (`asr_engine.js`)。
- **`config/`**: 配置中心，存放剪辑原则、用户偏好及 API 密钥。
- **`templates/`**: 结果展示模板。
- **`tests/`**: 自动化测试用例。

## 🛠️ 环境准备

1. **Node.js**: 确保已安装 Node.js 环境。
2. **FFmpeg**: 系统需安装 FFmpeg 以支持音频提取。
3. **API Key**: 在 `config/vendor.env` 中配置火山引擎的 AppID 和 Token。

## 📖 使用方法

1. **安装依赖**:
   ```bash
   npm install
   ```

2. **运行分析**:
   根据 Skill 触发词（如 "剪口播"）调用，工具将自动执行以下流程：
   - 提取音频
   - 语音转写 (ASR)
   - 智能分析问题片段
   - 输出剪辑建议

## ⚖️ 剪辑原则

详见 `config/principles/core.md`。主要包含：
- 过滤常见语气词（嗯、啊、这个、那个等）。
- 识别并标记超过 3 秒的长停顿。
- 捕捉重复表达的冗余内容。
